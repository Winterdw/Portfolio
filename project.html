Projetos Desenvolvidos:

  • Curso Cypress Intermediário
    Códigos de testes realizados na aplicação clone do Git Lab no intuito de aprofundamento no cypress através de testes de api's e de gui
  • Curso Cypress Básico
    Códigos de testes realizados numa aplicação básica de submissão de um formulário usando cypress
  • Trabalho de Conclusão de Curso: SegBee:
    Aplicativo desenvolvido com o objetivo de segmentar o mel das demais classes em uma placa de apiário através de uma foto retirada por um dispositivo móvel
  • Fighter's Marvel
    Versão do clássico Street Fighter compacta criada com os personagens da Marvel para estudos e aprimoração das habilidades de HTML, CSS e JS
  • Sistema de Notas Eletrônicas
    Sistema PHP com integração do banco de dados PostgreSQL, para criação, alteração e exclusão de notas eletrônicas sendo possível sua manipulação via navegador através de HTML, CSS e Javascript. Sua criação teve como objetivo treinamento de habilidades de Fullstack simulando uma parte de um sistema real.
  • Pokedex - Pokemón
    Versão compacta da Pokedex criada com o objetivo de aprimorar os conhecimentos de desenvolvimento Front End
  • Colônia de Formigas
    O trabalho consiste na implementação do algoritmo bioinspirado com o objetivo de realizar a análise do comportamento de uma colônia de formigas da vida real,tendo como uma heurística baseada em probabilidade, criada para solução de problemas computacionais que envolvem procura de caminhos em grafos. visando analizara busca de um alimento
  • Problema dos 5 Filósofos
    O problema consiste em cinco filósofos sentados ao redor de uma mesa circular para o jantar, onde cada filósofo possui um prato para comer macarrão. Os filósofos possuem hashis e cada um precisa de 2 hashis para comer, porém para cada par de pratos existe apenas um hashi. Um filósofo possui duas ações: pensar ou comer portanto a implementação da resolução do problema do jantar dos filósofos utilizando semáforos para controlar quando cada filósofo comerá
  • Trabalho de Classificação de Base de Dados
    O objetivo do trabalho foi a utilização de 4 algoritmos de Machine Learning (KNN (K-Nearest Neighbors), SVM(Support Vector Machine) Neural Network Perceptron e Naive Bayes) para a classificação de uma base de dados de comentários de filmes retirados do IMDB
  • Árvore B+ e Hash Extensível
    Este trabalho teve como objetivo a comparação entre dois algoritmos utilizando um Sistema de Gerenciamento de Banco de Dados (SGBD) utilizando a B+ e Hash Extensivel executando as operações de inserção, remoção e pesquisa em ambas estruturas.
  • Trabalho de Árvore B e Grafo de Redes de Fluxo
    O objetivo do trabalho foi a implementação de uma Árvore B completa com suas função principais (inserção, seleção, pesquisa e deleção) usando arquivo registro tipado e a implementação de Redes de Fluxo em um Grafo.
